# Полуавтоматические отклики на fl.ru

Система для полуавтоматических откликов на fl.ru: парсер присылает заказы в Telegram, по кнопке «Откликнуться» бот открывает заказ в **Firefox**, генерирует текст через ИИ, заполняет форму и прикрепляет резюме. **Отправку на сайте пользователь нажимает вручную.** После отправки пользователь нажимает в боте кнопку «Я отправил вручную -> Записать в отчет» — данные попадают в Excel.

## Как это работает

1. **Поиск:** Парсер (RSS) находит новые заказы и отправляет уведомления в Telegram.
2. **Запуск:** Пользователь нажимает кнопку **«✍️ Откликнуться»** в сообщении с заказом.
3. **Подготовка:** Бот открывает заказ в **Firefox** (Playwright), генерирует текст отклика через ИИ (`config/system_prompt.txt`), вставляет текст в форму и прикрепляет резюме. **Бот не нажимает «Отправить»** — вкладка остаётся открытой.
4. **Ручная отправка:** Пользователь проверяет текст, при необходимости решает капчу и нажимает «Отправить» на сайте.
5. **Запись в отчёт:** Пользователь нажимает в Telegram кнопку **«✅ Я отправил вручную -> Записать в отчет»** — в `report.xlsx` добавляется строка (ссылка, сопроводительное письмо, дата отклика, стоимость заказа).

## Требования

- Python 3.10+
- Файл `.env` (см. `.env.example`)
- Для ИИ: хотя бы один из ключей — OpenAI, Yandex GPT или GigaChat (выбор по минимальному ping)

## Установка

```bash
pip install -r requirements.txt
playwright install firefox
```

Перед первым запуском:

1. Скопируйте `.env.example` в `.env` и заполните `TOKEN`, `TELEGRAM_CHAT_ID`, при необходимости ключи ИИ и `RESUME_PATH`.
2. Положите резюме в `assets/resume.pdf` (или укажите свой путь в `RESUME_PATH`).
3. Инициализируйте БД: `python -m db.init_db`
4. Запуск: `python main.py`

## Структура проекта

- `config/system_prompt.txt` — системный промпт для ИИ (роль: Python-разработчик).
- `config/selectors.json` — CSS-селекторы для страницы fl.ru (поле отклика, файл резюме и т.д.).
- `assets/` — папка для файла резюме (`RESUME_PATH` в `.env`).
- `data/` — SQLite-база заказов (создаётся автоматически).
- `report.xlsx` — отчёт по откликам. Колонки: **Ссылка на заказ**, **Сопроводительное письмо**, **Дата отклика**, **Стоимость заказа**.
- `browser_profile/` — профиль Firefox (логин на fl.ru сохраняется здесь).

## Конфигурация

- **Telegram:** `TOKEN`, `TELEGRAM_CHAT_ID`.
- **Резюме:** `RESUME_PATH=./assets/resume.pdf` (если файла нет — в лог пишется предупреждение, отклики только текстом).
- **ИИ:** один или несколько из `OPENAI_API_KEY`, `YANDEX_API_KEY` + `YANDEX_FOLDER_ID`, `GIGACHAT_CREDENTIALS`. Используется провайдер с наименьшим ping.

Все комментарии в коде и сообщения бота — на русском языке.
